# Schiphol RxJava airport app
Projekt zbudowany przez Gradle wykorzystujący programowanie reaktywne na przykładzie tablicy lotów dla lotniska Schiphol w Amsterdamie.


W celu przetestowania właściwej funkcjonalności należy uruchomić Test1().
Dla podkreślenia braku zmian przy tych samych otrzymanych danych generowane są dodatkowo logi dotyczące odpytywania api.
